# 5号機 (ayase-web-service-5) Debian 13 + Proxmox VE セットアップレポート

- **実施日時**: 2026年2月25日 11:30 - 12:06

## 前提・目的

4号機 (ayase-web-service-4) と同一構成の Debian 13 + Proxmox VE 9 を5号機にセットアップする。
4号機で検証済みの os-setup スキル (Phase 1-8) をそのまま利用し、設定値 (hostname, BMC IP, static IP) のみ変更して実行した。

- 背景: 分散ストレージ (Ceph 等) の比較評価に複数ノードが必要
- 前提条件: 4号機のセットアップが完了済みであること、os-setup スキルが検証済みであること

## 再現方法

1. `config/os-setup.yml` の hostname, bmc_ip, static_ip を5号機用に変更
2. os-setup スキルの Phase 1-8 を順次実行
3. 完了後 `git checkout config/os-setup.yml preseed/preseed-generated.cfg` で設定復元
