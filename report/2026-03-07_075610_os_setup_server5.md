## 備考

- ISO ダウンロードは既存ファイル (sha256 一致) をスキップ
- ISO リマスターは preseed ハッシュ一致によりスキップ (前回と同一 preseed)
- POST code API はスケールの通り stale 値 (0x00/0x01 の繰り返し) を返す傾向があり、180 秒タイムアウトで進行
- VirtualMedia Boot ID は Boot0005 (ATEN Virtual CDROM)
- SOL 監視でインストール完了を検出 (Power down シグナル)
- SOL ログインで SSH 公開鍵、PermitRootLogin、静的 IP を設定
- PVE インストールは pre-reboot + post-reboot の 2 フェーズで完了
- cleanup フェーズの時間が長い (11m28s) のは最終リブート待ちを含むため
