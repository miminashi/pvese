# 8号機・9号機追加 + 7号機BMC IP変更 + リージョン再編レポート

- **実施日時**: 2026年3月19日 15:08 (JST)

## 前提・目的

### 背景

pvese プロジェクトは当初 4-6号機 (Supermicro X11DPU) + 7号機 (DELL R320) の4台構成で、LINSTOR/DRBD マルチリージョン運用を Region A (4+5号機) / Region B (6+7号機) の 2+2 構成で行っていた。

### 目的

1. 8号機・9号機 (DELL PowerEdge R320) を追加し、6台構成に拡張
2. 7号機の BMC IP を 10.10.10.120 → 10.10.10.27 に変更（連番体系に統一）
3. リージョン構成を 2+2 → 3+3 に再編（Region A: 4+5+6号機, Region B: 7+8+9号機）
4. 全ノード IB アダプタ搭載を反映し、Region B も IPoIB 通信に更新
5. PVE クラスタを 1リージョン1クラスタ構成に変更

### 前提条件

- 8号機・9号機が物理的に設置済みで、iDRAC から管理可能
- 全ノードに ConnectX-3 IB アダプタが搭載済み
- SX6036 IB スイッチに接続済み

## SSH config の手動対応手順

`~/.ssh/config` の更新が必要:

```
# 7号機: HostName を変更
Host idrac7
  HostName 10.10.10.27
  User claude
  IdentityFile ~/.ssh/idrac_rsa
  IdentitiesOnly yes
  KexAlgorithms +diffie-hellman-group14-sha1,diffie-hellman-group1-sha1
  HostKeyAlgorithms +ssh-rsa,ssh-dss
  PubkeyAcceptedAlgorithms +ssh-rsa

# 8号機: 新規追加
Host idrac8
  HostName 10.10.10.28
  User claude
  IdentityFile ~/.ssh/idrac_rsa
  IdentitiesOnly yes
  KexAlgorithms +diffie-hellman-group14-sha1,diffie-hellman-group1-sha1
  HostKeyAlgorithms +ssh-rsa,ssh-dss
  PubkeyAcceptedAlgorithms +ssh-rsa

# 9号機: 新規追加
Host idrac9
  HostName 10.10.10.29
  User claude
  IdentityFile ~/.ssh/idrac_rsa
  IdentitiesOnly yes
  KexAlgorithms +diffie-hellman-group14-sha1,diffie-hellman-group1-sha1
  HostKeyAlgorithms +ssh-rsa,ssh-dss
  PubkeyAcceptedAlgorithms +ssh-rsa
```

また、旧 IP のホスト鍵をクリア:
```sh
ssh-keygen -R 10.10.10.120
```
