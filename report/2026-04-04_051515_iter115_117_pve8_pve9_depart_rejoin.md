## 発見事項・注意点

1. **リソースあるノードの離脱手順**: ノード削除前に、そのノード上の全リソースレプリカを削除してから SP 削除が必要。手順: resource delete → storage-pool delete → node delete。
2. **auto-eviction 無効化のプロパティ名**: `AutoEvictAllowed` は無効。正しいキーは `DrbdOptions/AutoEvictAllowEviction`。
3. **再現性**: Iter 115 と Iter 117 (pve8 2回目) で完全に同一の手順が成功し、再現性を確認。
4. **pm-0c91e545 / pm-a2ef435f**: pve8/pve9 離脱後はこれらリソースが pve7 のシングルコピーになる。本評価では問題なし（DR 用リソースは別途再配置可能）。
