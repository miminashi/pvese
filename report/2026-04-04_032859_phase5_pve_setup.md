### Iteration 55: pve6 ブリッジ設定 + クラスタ参加

1. `/etc/network/interfaces` に vmbr0 (eno2np1, 10.10.10.206/8) + vmbr1 (eno1np0, DHCP) を設定
2. `systemctl restart networking` で適用、SSH 復帰確認
3. SSH 信頼関係構築 (pve4↔pve6 双方向)
4. `pvecm add 10.10.10.204 --use_ssh` でクラスタ参加成功

**結果**: pvese-cluster1 = pve4 + pve5 + pve6 (3/3, Quorate)

### Iteration 56: pve9 クラスタ参加

1. SSH 信頼関係構築 (pve7↔pve9)
2. **問題**: pve7 に pve9 の stale RSA 鍵が残存 → 新しい鍵に置換
3. `pvecm add 10.10.10.207 --use_ssh` でクラスタ参加成功

**結果**: pvese-cluster2 = pve7 + pve8 + pve9 (3/3, Quorate)

### Iteration 57: 最終検証

**クラスタ**: 両リージョン 3/3 Quorate。
