## 備考

- ISO は既存を再利用 (SHA256 一致)。preseed ハッシュファイルが無かったためリマスターは実行
- Phase 6: 初回 SSH 待機 (210s) がタイムアウト。preseed の `netcfg/choose_interface select auto` で選ばれたインターフェースと eno1 が異なるため、SOL 経由で eno1 に静的 IP を設定後に接続成功
- Phase 7: R320 POST は約 180 秒。リブート毎にデフォルトルートが 10.10.10.1 に戻るため `pre-pve-setup.sh` で修正
- RAID 構成: VD0 (RAID-1, /dev/sda, OS), VD1-VD4 (RAID-0, storage) — 変更なし
